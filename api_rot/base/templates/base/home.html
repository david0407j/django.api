Endpoints Disponíveis

  • GET /api/v1/resource: Retorna a lista de recursos.
  • POST /api/v1/resource: Cria um novo recurso.
  • GET /api/v1/resource/{id}: Retorna os detalhes de um recurso específico.
  • PUT /api/v1/resource/{id}: Atualiza um recurso específico.
  • DELETE /api/v1/resource/{id}: Exclui um recurso específico.
